# Personal Portfolio

A responsive and modern personal portfolio website built with React to showcase my skills, projects, and professional experience.

## Features
- **Responsive Design**: Optimized for all devices.
- **Dynamic Content**: Showcases skills, projects, and contact details.
- **Smooth Navigation**: Intuitive UI with transitions.
- **Contact Section**: Includes a direct contact form.

## Technologies Used
- React, HTML5, CSS3, JavaScript
- TailwindCSS/Chakra UI/Custom CSS (choose yours)
- Deployment: Vercel/Netlify

## Installation
1. Clone: `git clone https://github.com/kumarabhishek188/personal-portfolio.git`
2. Install: `npm install`
3. Run: `npm start` (Visit `http://localhost:3000`)

## Deployment
Live at: [https://abhishekportfolio12.netlify.app](https://abhishekportfolio12.netlify.app)
